Obviously, "<span class="tex-font-style-tt">1110011000</span>" is not good.</p><p>Tokitsukaze wants to make $$$s$$$ good by changing the values of some positions in $$$s$$$. all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, print a single line with one integer — the
